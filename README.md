# Farmer Rental Equipment and Solutions

## 📌 Description

The Farmer Rental Equipment and Solutions system is designed to help farmers easily rent agricultural equipment online. It connects equipment owners with farmers who need tools for a limited time, reducing cost and improving accessibility.

---

## 🚀 Features

* User registration and login
* Browse available farming equipment
* Rent equipment for specific duration
* Manage bookings and orders
* Admin management system
* Secure and user-friendly interface

---

## 🛠️ Technologies Used

* Frontend: React
* Backend: Spring Boot
* Database: MySQL

---

## 📂 Project Structure

```
Farmer-Rental-Equipment-and-Solutions
 ├── frontend
 ├── backend
```

---

## ▶️ How to Run the Project

### Backend (Spring Boot)

1. Open backend project in STS / Eclipse
2. Run the application
3. Server starts on:

```
http://localhost:8080
```

---

### Frontend (React)

1. Open frontend folder in VS Code
2. Install dependencies:

```
npm install
```

3. Start the application:

```
npm start
```

4. Runs on:

```
http://localhost:3000
```

---

## 🎯 Objective

To provide an efficient platform for farmers to rent equipment, reduce costs, and improve agricultural productivity.
